GoGoCha 隱私政策

最後更新：2026-04-21　｜　生效日：2026-04-21　｜　品牌：GoGoCha（花蓮計程車叫車服務）

GoGoCha（以下稱「本服務」）由個人開發者 Eric 營運，提供花蓮地區計程車叫車媒合服務。本政策說明我們如何蒐集、使用、儲存與保護您的個人資料，適用於本服務的 Android 應用程式（套件名稱：com.hualien.taxidriver）及相關網站。

1. 我們蒐集的資料

資料類型用途是否分享第三方
手機號碼Firebase Authentication 身份驗證（SMS OTP）否（僅用於驗證）
暱稱訂單上顯示給媒合對方否
精確位置（GPS）司機端：持續回傳供媒合；乘客端：叫車位置、路線規劃否
粗略位置（WiFi / 基站）GPS 無訊號時備援定位否
相機司機端拍攝里程表結帳憑證、大頭照否（僅上傳至本服務伺服器）
麥克風語音助理輸入地址（即時處理，不保存錄音）否
FCM Token訂單狀態推播通知否

2. 資料的使用目的

  • 媒合乘客與附近司機
  • 計算車資、規劃最短路線
  • 傳送即時訂單狀態（接單、到達、完成）
  • 客服查詢與糾紛處理
  • 系統穩定性與濫用防治

3. 第三方服務

本服務使用以下第三方服務處理部分功能。這些服務有各自的隱私政策，請視需要查閱：

  • Firebase Authentication（Google）— 手機號碼驗證
  • Firebase Cloud Messaging（Google）— 推播通知
  • Google Maps / Places / Directions API（Google）— 地圖顯示、地點搜尋、路線規劃

本服務不使用廣告 SDK、不販售使用者資料、不做跨網站追蹤。

4. 資料保留與刪除

  • 訂單紀錄保留 2 年供稅務與客服查詢
  • 位置紀錄僅於訂單進行中暫存，訂單完成後 30 天內刪除原始軌跡
  • 您可隨時於 App 內「個人資料」頁面登出；如需永久刪除帳號，請來信 26416387.re@gmail.com，我們將於 7 天內刪除所有可識別資料

5. 資料安全

  • 所有 API 通訊使用 HTTPS / TLS 加密
  • 手機端 token 透過 Android DataStore 儲存，並排除 Android Auto-Backup 避免雲端同步
  • 伺服器位於 AWS Lightsail，定期更新系統安全修補
  • 不儲存任何信用卡或支付資訊（車資現場交付司機，不經本服務）

6. 兒童隱私

本服務不針對 13 歲以下兒童設計，亦不主動蒐集兒童資料。若您為兒童的監護人並發現您的兒童使用本服務，請來信要求刪除。

7. 背景位置使用說明（Foreground Service Location）

司機端為即時媒合乘客，需在 App 進入背景時仍持續回傳位置。此功能僅在司機「上線接單」期間啟用，司機「下線」或關閉 App 後立即停止。前景服務會顯示常駐通知，讓使用者清楚知道位置正在被使用。

8. 政策變更

本政策如有重大變更將透過 App 內通知告知。繼續使用本服務即視為同意更新後之政策。歷史版本可來信索取。
